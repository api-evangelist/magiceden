# Magic Eden

Magic Eden is a multi-chain NFT marketplace and trading platform supporting Solana, Bitcoin (Ordinals and Runes), Ethereum, and Polygon. It provides REST APIs for accessing NFT listings, collection statistics, wallet activities, offers, and token metadata across supported chains, as well as trading instruction endpoints to generate and submit buy, sell, bid, and cancel orders directly on the Magic Eden marketplace.

**Developer Portal:** https://docs.magiceden.io  
**API Reference:** https://docs.magiceden.io/reference/solana-overview  
**API Terms of Service:** https://magiceden.us/legal-policies/api-terms  
**GitHub:** https://github.com/magiceden

## APIs Cataloged

- **Solana Collections API** — Collection listings, stats, attributes, activities, holder stats, leaderboard
- **Solana Tokens API** — Token metadata, listings, offers, and activity by mint address
- **Solana Wallets API** — Wallet holdings, activities, escrow balance, offers made/received
- **Solana Trading Instructions API** — Buy, sell, bid, cancel, accept offer, escrow operations
- **Solana AMM Pool API** — Create/manage MMM liquidity pools, fulfillment, best-offer queries
- **Bitcoin Ordinals API** — Inscriptions, collection stats, rare sats (deprecated March 2026)
- **Runes API** — Rune activities, orders, swaps, quotes, bulk sweeping (deprecated March 2026)
